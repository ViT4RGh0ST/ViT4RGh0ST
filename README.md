## 🚀 What I've Built

### 🏪 Production POS Platform — *5 years, 12-dev team*
> Core engineer for a US furniture retailer's e-commerce and POS system

- 🔧 **Own the service layer**: orders, quotes, inventory (ATP), payments, refunds, tax, delivery
- 🔌 **SAP B1 integration architect**: OData, MCP, audit logging, feature flags
- 💳 **Clover payment hardware**: Flex devices, REST Pay Display API, in-store transactions
- 🖥️ **35+ Vue 3 components** powering the POS interface
- ✅ **200+ tests** covering critical business flows
- 📊 2nd most active contributor out of 12 developers

`Laravel 13` `PHP 8.4` `Vue 3` `Inertia.js` `SAP B1` `Clover API` `Filament` `Pest`

### 🤖 Landifly — *AI SaaS, solo, 37 days*
> Users chat → AI generates landing pages → one-click deploy

- 🧠 Custom AI agent with **10 tools**, SSE streaming, multi-model routing (OpenRouter)
- 🚀 Multi-provider deploy: **Netlify / Vercel / Cloudflare Pages** via factory pattern
- 💰 Complete SaaS billing: **6 tiers, 12 feature gates, credit packs** (LemonSqueezy)
- 📝 **106 documented engineering decisions** with analysis and trade-offs
- 🔒 Security hardened: prompt injection guards, rate limiting, HTML sanitization
- 🌐 Bilingual (EN/ES), dark mode, A/B testing

`Laravel 12` `Vue 3` `TanStack Query` `OpenRouter` `LemonSqueezy` `Cloudflare R2` `Redis`
